# Movie-Ticket-Booking-System
A full-stack Movie Ticket Booking System that allows users to browse movies, view show timings, select seats, and book tickets online. The project demonstrates real-world web application features such as authentication, CRUD operations, seat management, and database integration. Built to showcase practical skills in Python Full Stack Development.
